# SaaS Growth & Unit Economics Dashboard

## Overview

An interactive Tableau dashboard analyzing SaaS growth performance through customer acquisition, revenue, and unit economics.

## Objectives

- Monitor CAC
- Analyze LTV
- Measure ROAS
- Evaluate conversion performance
- Track business growth

## Tools

- Tableau Public
- Microsoft Excel

## Key Results

- Total Customers: **150**
- Total MRR: **$23,600**
- Average CAC: **$42.63**
- LTV: **$786.67**
- LTV/CAC Ratio: **18.5x**
