# SaaS Subscription Revenue Analysis

## Overview

This project analyzes SaaS subscription data to evaluate revenue performance, customer retention, acquisition channels, and churn using Microsoft Excel.

## Objectives

- Analyze Monthly Recurring Revenue (MRR)
- Measure customer retention and churn
- Identify top-performing customer segments
- Evaluate acquisition channel performance

## Tools

- Microsoft Excel
- Pivot Tables
- XLOOKUP
- Dashboard

## Key Results

- MRR: **$5,461**
- Retention Rate: **78%**
- Churn Rate: **22%**
- Top Revenue Segment: **Mid-Market**
- Best Acquisition Channel: **Google Ads**
