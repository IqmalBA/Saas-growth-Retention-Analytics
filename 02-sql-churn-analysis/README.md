# SaaS Customer Churn & Revenue Impact Analysis

## Overview

This SQL project investigates customer churn, recurring revenue loss, and customer value to support retention strategy.

## Objectives

- Measure customer churn
- Analyze revenue loss
- Identify high-risk customer segments
- Calculate ARPU

## Tools

- PostgreSQL
- SQL

## Key Results

- Churn Rate: **20%**
- Revenue Lost: **$9,284 / month**
- Highest Churn: **Enterprise & SMB**
- Highest ARPU: **Enterprise**
